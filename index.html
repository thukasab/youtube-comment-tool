⚡主な機能

🔑

シンプルな操作

API キーと YouTube URL を入力して「取得開始」を押すだけ。複雑な設定は不要です。

📦

コメント全件取得

ページネーションを自動処理し、コメントを漏れなく全件取得。取得件数はリアルタイムで表示。

📊

Excel 出力

7カラム（コメントID / 投稿者名 / 日時 / 本文 / いいね数 / 返信数）で整形出力。列幅も自動調整。

💾

API キー自動保存

入力した API キーをローカルに保存。次回起動時に自動で復元されます。

🔗

複数 URL 形式に対応

通常の動画・短縮 URL（youtu.be）・Shorts の 3 形式すべてに対応。

📦

exe 配布可能

PyInstaller で単体 exe に変換済み。Python 不要で納品できます。

⚙️動作環境・仕様

対応 OSWindows 10 / 11
実行形式単体 exe（Python インストール不要）／ Python スクリプト（3.9 以上）
必要なものYouTube Data API v3 の API キー（Google Cloud Console で無料取得）
API 制限無料枠（1 日 10,000 ユニット）で通常利用は十分
出力形式.xlsx（Excel）/ 列幅自動調整・ヘッダー色付き
対応 URLyoutu.be / youtube.com/watch / youtube.com/shorts

🛠技術スタック

Python 3.9+ Tkinter（GUI） YouTube Data API v3 google-api-python-client openpyxl PyInstaller（exe 化）
